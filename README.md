# chn-ssl-guide

本仓库用于归档与发布多个独立 HTML 页面，内容主要涉及与 SSL/TLS 相关的通用信息与参考指南。仓库不针对任何单个域名或具体网站，所有页面均为独立发布，不构成对任何特定服务或产品的推荐或背书。

## 项目简介

本项目旨在收集并整理与 SSL/TLS 相关的说明性 HTML 页面，以方便查阅与引用。所有内容以静态 HTML 形式存放，适合直接部署或离线浏览。

## 目录说明

- `pages/` — 存放所有独立 HTML 页面，每个页面对应一个独立主题。
- `assets/` — 存放页面所需的静态资源（如样式表、图片等）。
- `index.html` — 仓库首页，提供页面索引与导航。

## 页面归档说明

- 每个 HTML 页面均为完整自包含文件（或依赖 `assets/` 目录下的资源）。
- 页面内容以通用技术说明为主，不包含具体域名或营销信息。
- 归档页面可能不定期更新，但不对更新频率做任何承诺。

## 维护说明

- 本仓库由维护者根据个人兴趣与时间进行更新。
- 欢迎提交 issue 或 pull request 进行内容修正或补充，但需确保内容符合上述通用性原则。
- 不接受任何涉及具体商业推广或定向引导的内容。

## 使用方式

可直接通过 GitHub Pages 或其他静态托管服务部署本仓库内容。如需本地浏览，克隆仓库后使用任意 HTTP 服务器打开 `index.html` 即可。

```bash
git clone https://github.com/your-username/chn-ssl-guide.git
cd chn-ssl-guide
# 使用 Python 内置服务器示例
python3 -m http.server 8000
```
